Reading Stats for PocketBook - install notes

Tested target: PocketBook InkPad Color 3 / SDK 6.8 build.

Files from the GitHub Actions artifact:
- ReadingStats.app
- ReadingStats.app.bmp
- ReadingStats.bmp
- ReadingStats_f.bmp
- install-pocketbook.txt

Copy the app file to the PocketBook applications directory:

  applications/ReadingStats.app

For launcher icons, create this directory on the device:

  applications/icons/

Then copy icon files from the artifact as:

  applications/icons/ReadingStats.bmp
  applications/icons/ReadingStats_f.bmp

The compatibility bitmap can also be copied next to the app while testing:

  applications/ReadingStats.app.bmp

Launcher registration / name and icon:

PocketBook custom applications can be registered in the launcher view config as
user apps. User app ids must start with U_. The launcher config is usually here:

  /system/config/desktop/view.json

Keep a backup of the original view.json before editing it. Invalid JSON may show
an error popup or break the Applications menu until the file is restored.

Recommended icon layout based on KOReader customization notes:

  applications/icons/ReadingStats.bmp
  applications/icons/ReadingStats_f.bmp

Icon requirements seen in PocketBook customization examples:

  BMP format
  8-bit color depth
  tested max around 106 px wide and 128 px high

Recommended ReadingStats entry using absolute paths:

  "U_ReadingStats": {
    "path": "/mnt/ext1/applications/ReadingStats.app",
    "title": "Statystyki czytania",
    "icon": "/mnt/ext1/applications/icons/ReadingStats.bmp",
    "focused_icon": "/mnt/ext1/applications/icons/ReadingStats_f.bmp"
  }

Then add U_ReadingStats to one of the view groups, for example @General:

  "apps": [
    "U_ReadingStats",
    "PB_AudioBooks",
    "PB_Browser"
  ]

Name-only fallback:

If the app still appears as @ReadingStats, the display name can also be mapped
through the language file matching your UI language, for example:

  /system/language/pl.txt

Add a line like:

  @ReadingStats=Statystyki czytania

Runtime directory kept intentionally unchanged:

  /system/pbreadstats/

Optional config file:

  /system/pbreadstats/config.cfg

Useful config values:

  language=pl
  daemon=1

Safe diagnostic mode:

  daemon=0

Debug logging:

  debug=1

or create this trigger file:

  /system/pbreadstats/LOGTRIGGER.TXT

Generated runtime files:

  /system/pbreadstats/reading_stats.db
  /system/pbreadstats/debug.log
  /tmp/pbreadstats.pid

Clean uninstall for testing:

  Remove applications/ReadingStats.app
  Remove applications/ReadingStats.app.bmp
  Remove applications/ReadingStats.bmp
  Remove applications/icons/ReadingStats.bmp
  Remove applications/icons/ReadingStats_f.bmp
  Remove /system/pbreadstats/
  Remove U_ReadingStats from the launcher view config if you added it.
  Remove @ReadingStats from the language file if you added it.

Controls on InkPad Color 3:

  Touch middle-left  = previous screen
  Touch middle-right = next screen
  Touch center       = refresh data
  Physical left key  = previous screen
  Physical right key = next screen
  Home               = exit app
